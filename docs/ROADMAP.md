# Roadmap prodotto

## Fase 1 — MVP locale

- Single page app statica.
- Admin JSON per tutti i minigiochi.
- Presentazione live.
- Scoreboard e correzioni manuali.
- Import/export JSON.
- Supporto media da cartella `public/assets/`.

## Fase 2 — Creator più comodo

- Form visuali dedicati per ogni minigioco.
- Upload media dall'interfaccia admin.
- Validazione schema con errori leggibili.
- Duplicazione round/domande più rapida.
- Anteprima media nel pannello admin.

## Fase 3 — Modalità show

- Vista presentatore e vista pubblico separate.
- Timer configurabile per round.
- Effetti sonori e animazioni reveal.
- Layout fullscreen ottimizzato per TV/proiettore.
- Tema grafico aderente al PowerPoint originale.

## Fase 4 — Multiplayer reale

- Login host.
- Lobby partita.
- Join da smartphone tramite QR code.
- Buzzer per Jeopardy e risposte rapide.
- Salvataggio partite su database remoto.
- Storico classifiche e statistiche.

## Fase 5 — Produzione

- Backend con Supabase/Firebase o API Node + PostgreSQL.
- Ruoli: owner, editor, host, player.
- Storage media con CDN.
- Test end-to-end.
- CI/CD completo.
- Backup automatici.
